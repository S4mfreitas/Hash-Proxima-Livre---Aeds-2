# Hash-Pr-xima-Livre---Aeds-1
Trabalho de AEDS II (UNIMONTES). Implementação de estrutura de indexação via Tabela Hash (Próxima Livre) comparada à Busca Sequencial, utilizando o dataset "NBA Stats". O sistema carrega registros em memória e analisa o desempenho das consultas. Autores: Luís Felipe e Samuel Freitas.
